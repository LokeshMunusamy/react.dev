title: Light - Dark Theme
---

Dark mode displays more visually appealing, and they can reduce power consumption

### How to change dark/light theme? {/*how-to-change-darklight-theme*/}

1. Login to QA Touch.
2. In the top right, tap your profile picture.
3. Click Edit Profile.
4. Select the theme (Dark / Light) Mode.
