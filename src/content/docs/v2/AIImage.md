### AI Test Case Generation {/*ai-test-case-generation*/}

QA Touch now allows testers to generate test cases directly from FIGMA or screenshots or UI mockups using AI. This saves time and boosts accuracy by reducing manual test case writing effort.

Steps:

To create Test Cases from Images in QA Touch:

1. Go to the Test Cases page of your project
2. Click on Add Test Case -> AI Image Description Template
3. Upload a UI screenshot or mockup image
4. Click on Generate Test Case using AI
5. QA Touch will automatically analyze the image, identify key UI elements (buttons, forms, menus), and suggest meaningful test cases
6. Review and refine the AI-generated test cases before saving
