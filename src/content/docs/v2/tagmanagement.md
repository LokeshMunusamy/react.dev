title: How to use Tag Management
---

### Add New Tag {/*add-new-tag*/}

1. Go to Administration-> Custom option
2. Go to the Test Cases tab and click on Tags
3. Add new Tags here

![tag_managment help1 image](../../assets/faq-screenshots/tag_managment/help1.png)

### Adding / Updating New Tags to Test case {/*adding--updating-new-tags-to-test-case*/}

1. Adding/updating Tags can be done using the Add / Edit Test case

![tag_managment help2 image](../../assets/faq-screenshots/tag_managment/help2.png)

### Bulk update of Tags {/*bulk-update-of-tags*/}

1. Check the required Test cases, and use the Bulk update option in the Test case List page

![tag_managment help3 image](../../assets/faq-screenshots/tag_managment/help3.png)

### Test Case search {/*test-case-search*/}

1. Search using New Tags introduced in the search page

![tag_managment help4 image](../../assets/faq-screenshots/tag_managment/help4.png)

### Bulk add/ remove Tags in the search page {/*bulk-add-remove-tags-in-the-search-page*/}

1. Select the required Test cases in the search page
2. Click on Tags in accordion
3. Choose the required Tag
4. Click on Add to add the tag to all selected Test cases / Click on Remove to remove the tag from all selected Test cases

![tag_managment help5 image](../../assets/faq-screenshots/tag_managment/help5.png)
