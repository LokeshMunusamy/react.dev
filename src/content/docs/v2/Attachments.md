---
title: Attachments
---

Attachments are any project related files such as Business Description Document(BDD), plan and so on. It helps project team to understand the requirements, deliverables and activities to work upon.


You can add any file attachment to QA Touch by going following steps:

1. Login to QA Touch.
2. On the left panel, click on Attachments.

You can attach file types highlighted in the below image.

**Note:** A single file size should be less than 2MB and only a maximum of 10 files are allowed to be attached in a project.
